=========
Examples
=========
End-to-end examples show how you can use PyTwin. If the PyTwin ``ansys-fluent-core`` package is installed on your machine,
you can download these examples as Python files or Jupyter notebooks and run them locally.

=========================
These examples show how to use the evaluation workflow on twins. This includes simple evaluation workflow, as well as parametric applications for static and dynamic models.
